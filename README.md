CodeIn is a modern, vibrant, frontend-only code playground that allows users to write and execute **HTML, CSS, and JavaScript directly in the browser**.  
It is inspired by tools like CodePen and JSFiddle and is built using **pure web technologies**, with no backend or API integrations.

##  Features
-  Separate editors for **HTML**, **CSS**, and **JavaScript**
-  Run button to execute code instantly
-  Live preview rendered using a sandboxed iframe
-  Custom console panel that displays `console.log()` output
-  Vibrant dark UI with glowing neon design
-  Fully client-side (no backend, no database, no APIs)
-  Deployed using **GitHub Pages**

---

##  Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **GitHub Pages** (for deployment)

---

##  How It Works
The editor dynamically injects user-written HTML,CSS and JavaScript into an iframe using the srcdoc property.JavaScript execution is intercepted to capture console.log() output and display it in a custom console panel, enabling real-time feedback inside the UI.
